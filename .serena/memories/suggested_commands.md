# Suggested Commands

## Development
```bash
npm run dev          # Start dev server (localhost:5173)
npm run build        # TypeScript check + production build
npm run preview      # Preview production build
```

## Testing & Quality
```bash
npm run test         # Run Jest tests (uses experimental ESM)
npm run lint         # ESLint check
npm run format       # Prettier format src/
```

## Serena
```bash
npm run serena       # Start serena server
npm run serena:start # Start MCP server
npm run serena:index # Index project
```

## System Commands (Windows)
Since this is Windows, use these equivalents:
- `dir` instead of `ls` (or use Git Bash/PowerShell `ls`)
- `type` instead of `cat`
- `findstr` instead of `grep`
- `where` instead of `which`
- Git Bash provides unix-like commands in this environment
