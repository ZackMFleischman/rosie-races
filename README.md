A fun racing game for 4-year-old Rosie featuring math challenges!

## Overview

Race against family members (Mommy, Daddy, Uncle Zack, Gaga, Grandpa, and Lalo the dog) by tapping to run. Stop at checkpoints to solve math problems for speed boosts!

## Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Game Engine:** Phaser 3
- **UI:** MaterialUI
- **Platform:** PWA (installable on iPhone/iPad)
- **Hosting:** GitHub Pages

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run lint` | Run ESLint |
| `npm run test` | Run Jest tests |
| `npm run format` | Format with Prettier |
